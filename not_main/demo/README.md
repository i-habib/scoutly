# Demo Code Archive

This folder contains the original TanStack Start demo code for reference purposes.

## Contents

- **routes/** - Demo route components showing TanStack Start features
  - `api.names.ts` - API route example
  - `start.api-request.tsx` - API request demonstration
  - `start.server-funcs.tsx` - Server functions example
  - `start.ssr.*.tsx` - Various SSR mode demonstrations

- **data/** - Demo data files
  - `demo.punk-songs.ts` - Sample data used in demos

## Note

These files are kept for reference but are not part of the active Scoutly application. They demonstrate various TanStack Start patterns that may be useful during development.
